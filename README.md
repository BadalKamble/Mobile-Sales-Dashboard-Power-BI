📌 Project Title
Mobile Sales Dashboard — Brand, City & Payment Analytics | Power BI

📝 Short Description
An interactive Power BI dashboard analyzing mobile phone sales across India — tracking revenue, quantity, transactions, customer ratings, and payment methods. Covers 5 major brands across 20+ Indian cities with month-wise and day-wise drill-down filters for deep business insights.

🎯 Purpose
This project was built to:

Track mobile sales performance across brands — Samsung, OnePlus, Xiaomi, Vivo, Apple
Analyze city-wise sales distribution across 20+ Indian cities using map visualization
Monitor payment method trends — Debit Card, Credit Card, Cash, UPI
Identify top-performing mobile models and best sales days
Practice Power BI dashboard design and DAX measures for BA portfolio
